# ad-rest-api

## Pre-requisites

1. `docker pull dwimberger/ldap-ad-it`

## Running locally

In a terminal window: 
1. `make start-ad`

In a new terminal window: 

1. `make start`

## Running tests

In a terminal window: 
1. `make start-ad`

In a new terminal window: 

1. `make test`
2. `make test-cover` to run unit test coverage
